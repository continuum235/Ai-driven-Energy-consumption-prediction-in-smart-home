# Dataset Directory

Place the UCI Individual Household Electric Power Consumption dataset
file (`data.txt`) in this directory before running the pipeline.

## Download

- **Source:** [UCI ML Repository](https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption)
- **File:** `household_power_consumption.txt` → rename to `data.txt`

## Format

- Semicolon-separated values
- Columns: `Date;Time;Global_active_power;Global_reactive_power;Voltage;Global_intensity;Sub_metering_1;Sub_metering_2;Sub_metering_3`
- Date format: `dd/mm/yyyy`
- Time format: `HH:MM:SS`
- Missing values marked with `?`
